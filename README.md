# AI Star Launchpad

AI Star Launchpad is a Codex skill for turning a short AI product prompt into a repo package that is easier to ship, demo, and share.

## What It Produces

- repo name and one-line pitch
- target user and MVP scope
- recommended stack and file tree
- README opening and landing page copy
- demo flow and launch checklist

## Good Fit

- AI developer tools
- vertical AI apps
- agent workflow products
- existing repos that need clearer positioning

## Example Prompts

- Build an AI PR review tool
- Build an MCP debugger
- Upgrade this repo so it is easier to launch

## Repository Layout

- `SKILL.md`: main instructions
- `references/`: heuristics and templates
- `scripts/`: generators and scaffold helpers
- `agents/`: supporting agent material

## Scope

This repo is for repo planning and launch packaging. It focuses on fast demos, clear positioning, and practical deliverables.
